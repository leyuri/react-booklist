# react-booklist
경제 경영 관련 서적 리스트를 확인할 수 있는 웹앱! 📚

### Development environment
- react.js
- material-ui
- redux, react-redux, immer

### Features
- 선택된 책에 맞는 책 상세 설명을 보여준다. 
- 책을 추가할 수 있다. 
- 선택된 책에 대해 좋아요 버튼을 누를 수 있다. (좋아요 +)

### Components
- BookDetail : 책의 상세 내용
- BookList : 책 리스트
-TopNav : 탑 메뉴바
